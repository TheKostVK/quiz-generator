# Quiz Generator

Приложение для создания, сохранения и прохождения квизов на основе JSON-описания. Поддерживает одиночный и множественный выбор, валидацию структуры квиза и хранение тестов в IndexedDB.

## Демо
Опубликованная версия: **https://thekostvk.github.io/quiz-generator/**

## Технологии

- **Vite** — сборка и dev-сервер
- **Vanilla JavaScript (ES Modules)** — логика приложения
- **Zod** — валидация структуры квиза
- **IndexedDB** — хранение квизов в браузере
- **HTML5 / CSS3** — разметка и стили
- **БЭМ** — методология именования классов

## Формат квизов

Квиз описывается в формате JSON следующей структуры:

```json
{
  "title": "Основы HTML",
  "description": "Проверьте свои знания по HTML: теги, атрибуты и семантика",
  "questions": [
    {
      "id": 1,
      "text": "Какой тег используется для создания ссылки?",
      "type": "single",
      "options": [
        {
          "id": 1,
          "text": "<link>",
          "correct": false,
          "message": "Тег <link> используется для подключения внешних ресурсов"
        },
        {
          "id": 2,
          "text": "<a>",
          "correct": true,
          "message": "Верно! Тег <a> создает гиперссылки"
        }
      ]
    }
  ]
}
```
## Структура квиза

### Поля квиза

- `title` — название квиза
- `description` — описание квиза
- `questions` — массив вопросов

### Структура вопроса

Каждый вопрос содержит:

- `id` — уникальный идентификатор вопроса
- `text` — текст вопроса
- `type` — тип вопроса: `single` или `multiple`
- `options` — массив вариантов ответа

### Структура варианта ответа

Каждый вариант ответа содержит:

- `id` — уникальный идентификатор варианта
- `text` — текст варианта
- `correct` — является ли вариант правильным (`true` / `false`)
- `message` — пояснение для пользователя
